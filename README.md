# Solo Command

A personal productivity tool for solo developers juggling multiple projects. Track daily standups, capture ideas, and visualize your work - all without leaving your browser.

## Features

- **Daily Standups** - Track what you accomplished, what's next, and any blockers
- **Project Management** - Organize work by projects with tagging and archiving
- **Board View** - Kanban-style board showing individual tasks organized by project
- **Timeline View** - Chronological view of all your entries
- **Ideas** - Capture and link ideas to projects
- **Demo Data** - Generate fake data to test the app

## ⚠️ Local Storage Notice

**All data is stored locally in your browser using localStorage:**
- ✅ Your data is completely private
- ✅ No sign-up or login required
- ✅ Works offline
- ⚠️ Data is device-specific (won't sync across browsers/devices)
- ⚠️ Clearing browser data will delete your entries

## Deploy to Vercel

This app is ready to deploy to Vercel with zero configuration:

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

That's it! No environment variables or database setup needed.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Client-side localStorage (no backend/database)
